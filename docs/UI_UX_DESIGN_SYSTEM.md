# Lumina AI OS — UI/UX Design System

## Brand
- Name: Lumina AI OS
- Tagline: "The Autonomous Digital Employee"
- Voice: Professional, clear, helpful

## Color Palette
| Role | Color | Hex |
|------|-------|-----|
| Background | Dark Navy | #030712 |
| Surface | Gray-900 | #111827 |
| Border | Gray-800 | #1F2937 |
| Primary | Cyan-400 | #22D3EE |
| Primary Hover | Cyan-500 | #06B6D4 |
| Text Primary | Gray-100 | #F3F4F6 |
| Text Secondary | Gray-400 | #9CA3AF |
| Success | Green-400 | #4ADE80 |
| Warning | Yellow-400 | #FACC15 |
| Error | Red-400 | #F87171 |

## Typography
- Font: system-ui, -apple-system, sans-serif
- Headings: Bold, white
- Body: Regular, gray-100
- Code: monospace, smaller size

## Spacing
- Base unit: 4px
- Padding: 16px (cards), 24px (sections)
- Gap: 16px (between cards)
- Border radius: 12px (cards), 8px (buttons/inputs)

## Components

### Cards
- Background: gray-900 (#111827)
- Border: gray-800 (#1F2937)
- Border radius: 12px
- Padding: 20px

### Buttons
- Primary: Cyan-600 bg, white text
- Secondary: Gray-800 bg, gray-300 text
- Small: 8px padding, 12px border-radius
- Large: 12px padding, 8px border-radius

### Inputs
- Background: gray-800
- Border: gray-700
- Border radius: 8px
- Padding: 8px 12px
- Text: gray-200

### Navigation
- Width: 256px (sidebar)
- Background: gray-900
- Active item: cyan-600/20 bg with cyan-300 text

## Layout
- Full-height sidebar navigation
- Content area with scroll
- Responsive grid (1/2/3/4 columns)
- Max-width: 1200px for content

## Dark Mode
Always dark — Lumina AI OS is designed for extended professional use in low-light environments.
